NAME

ARLE ‚ÄĒ a Rust-native inference runtime with integrated local agent, train, and eval workflows. CUDA is the primary serving path; Metal and CPU smoke paths share the same workspace, model code, and tooling surface.

This site is intentionally a stable landing page. Detailed product facts live in the repo docs linked below.

SYNOPSIS

$ docker run --gpus all -v /path/to/Qwen3-4B:/model \
    ghcr.io/cklxx/agent-infer:latest --model-path /model --port 8000
$ cargo build -p infer --release
$ ./target/release/infer --model-path /path/to/Qwen3-4B --port 8000
$ cargo build -p agent-infer --release --features cli --bin arle
$ ./target/release/arle --model-path /path/to/Qwen3-4B --max-turns 10

STATUS

CUDAStable primary serving path on Linux + NVIDIA.
MetalBeta Apple Silicon path with live runtime, DFlash, and the MLX bridge.
CPUDevelopment-only smoke path for request-surface validation.
Bench claimsDated snapshots live under docs/experience/wins/; active CUDA closure work lives in the current decode-alignment plan.

QUICKSTART

— Source build

$ git clone https://github.com/cklxx/arle
$ cd arle
$ ./setup.sh
$ cargo build -p infer --release
$ ./target/release/infer --model-path /path/to/Qwen3-4B --port 8000

— API smoke

$ curl http://localhost:8000/v1/chat/completions \
    -H 'Content-Type: application/json' \
    -d '{"messages":[{"role":"user","content":"Hello"}],"max_tokens":64}'

Use the repo-pinned Rust toolchain from rust-toolchain.toml. ./setup.sh bootstraps Rust, Python, Zig, and local checks.
