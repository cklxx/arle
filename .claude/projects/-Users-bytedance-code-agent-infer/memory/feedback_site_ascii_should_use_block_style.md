# Rule

The landing-page ASCII hero should use a block-style wordmark that spans wide,
not a thin line-art wordmark.

# Why

The user explicitly corrected the hero direction: it should keep the earlier
filled/block feel and visually occupy the width, instead of switching to a
narrow outline treatment.

# Preventive action

- Prefer filled block glyphs like `█`/`╔`/`╝` over line-only slash outlines.
- Keep the `ARLE` wordmark visually wide enough to anchor the whole hero.
- Treat the warm man-page layout as stable; only adjust the hero text style.
