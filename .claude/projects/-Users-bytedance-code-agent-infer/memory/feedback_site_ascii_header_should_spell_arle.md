# Rule

The landing-page ASCII hero must spell `ARLE`.

# Why

The user explicitly said the ASCII header is intentional design, and then
corrected the content: it should show `ARLE`, not the older `AGENT INFER`
lettering.

# Preventive action

- Keep the ASCII hero block on `site/index.html` and `site/zh-CN.html`.
- When updating site copy, change only the text inside that design, not the
  surrounding layout.
- Treat the current `ARLE` ASCII wordmark as the canonical landing-page hero.
